# PROJECT-BIOINFORMATIKA_REGULER-25_2026
Pada project ini, kami melakukan metode Molecular Docking dan analisis ADMET (Adsorption, Distribution, Metabolism, Excretion, and Toxicity) terhadap beberapa senyawa metabolit yang terkandung pada daun Psidium guajava yang akan diujikan ke protein target berupa NS2B/NS3 dari Virus Dengue (DENV)
